macOS에 JDK 설치하기

2018-04-08
java, jdk

1. JDK 다운로드

JDK는 오라클(http://www.oracle.com/)에서 제공합니다. 아래의 경로에 접속하면 최신버전의 JDK를 다운로드 받을 수 있습니다.

JDK download URL : http://www.oracle.com/technetwork/java/javase/downloads/index.html

혹은 오라클 사이트에서 menu > Databases and Technologies > Developer Tools > Java SE SDK 를 선택하고 download 탭을 클릭합니다.

위 화면에서 Java Download 이미지를 클릭합니다.

Accept License Agreement에 체크하고 macOS 용 dmg 파일을 클릭하면 다운로드가 진행됩니다.

2. JDK 설치

dmg 형식으로 제공되기 때문에 클릭 몇번으로 쉽게 설치를 할 수 있습니다.

상자 모양 아이콘을 더블클릭하여 설치를 진행합니다.

3. JDK 설치 확인

macOS에 JDK를 설치하면 아래 경로에 설치됩니다.

위 경로로 이동해서 디렉토리 목록을 확인해보면 JDK 버전별로 디렉토리가 생성되는걸 볼 수 있습니다.

혹은 /usr/libexec/java_home 명령어를 실행하여 설치된 JDK의 home path를 확인할 수 있습니다.

여기에 -V옵션을 추가하면 설치된 모든 JDK의 목록을 볼 수 있습니다.

저는 8개의 JDK 목록이 나오는군요.

혹은 java -version 명령어를 통해 자바가 정상적으로 설치되고 path가 잡혀있음을 확인할 수 있습니다.
